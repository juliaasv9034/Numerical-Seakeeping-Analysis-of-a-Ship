# Numerical Seakeeping Analysis of a Ship (Flokstra Method)

This repository contains a comprehensive seakeeping analysis developed at **Instituto Superior Técnico (IST)**. The project focuses on predicting the dynamic response of a vessel (specifically the **Flokstra** tanker hull) subjected to regular and irregular seas.

## Project Overview
The study involves the calculation of ship motions in the frequency domain, focusing on **Heave** and **Pitch** coupling. The analysis evaluates how the hull geometry interacts with wave-induced forces and the resulting hydrodynamic coefficients.

## Key Technical Aspects
* **Hydrodynamic Coefficients:** Calculation of added mass, damping, and restoring force coefficients.
* **RAO Generation:** Development of Response Amplitude Operators (RAOs) for different Froude numbers and heading angles.
* **Wave Spectrum Analysis:** Application of the **Pierson-Moskowitz** (or JONSWAP) spectrum to evaluate short-term responses in irregular waves.
* **Motion Prediction:** Assessment of Vertical Relative Motion (VRM) and the probability of slamming or deck wetness.
* **Numerical Method:** Implementation based on the **Flokstra tanker** data and strip theory principles.

## Tools Used
* **Rhino/ Excel:** For numerical integration and solving the equations of motion.
* **Technical Writing:** Comprehensive report detailing the methodology, results, and sea state analysis.
